Through hands-on projects and real-world workflows, I developed practical skills in front-end development, version control, debugging, and site deployment.

  • Structuring accessible webpages with semantic HTML

  • Using Git & GitHub for version control—repositories, commits, branches, pull requests, and issues

This course strengthened my ability to deliver clean, responsive, and well-documented front-end projects. I’m confident applying these skills to real-world web projects and continuing to expand my front-end toolkit.
